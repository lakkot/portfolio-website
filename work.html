Approval Tool

Multi-level approval tool for documents, performed via email without the need for the approvers to create accounts.
User first creates lists of approver emails, then uploads a document, adds description and chooses approvers. Documents are then sent sequentially to the approvers via email. Operated as subscription service. Utilizes Amazon S3 bucket and braintree payments. Written in MERN (MongoDB, Express JS, React and Node JS).
